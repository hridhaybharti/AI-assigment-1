# 🤖 AI Chat Bot - Assignment 1

A professional, conversational AI chat bot ecosystem designed for high-performance natural language interactions. This project features a robust Node.js backend and a clean, responsive web interface for seamless AI communication.

## 🚀 Overview

The **AI Chat Bot** provides an interactive platform for real-time conversational intelligence. It's built with focus on low-latency responses, clean architectural separation between client and server, and an intuitive user experience.

## 🛠️ Tech Stack

- **Backend:** Node.js, Express (Modular Middleware Architecture)
- **Frontend:** HTML5, CSS3 (Modern Responsive Design), JavaScript
- **Communication:** RESTful API for stateless interaction
- **Server:** High-concurrency Node.js environment

## 📡 Key Features

- **Real-time Processing:** Synchronous and asynchronous message handling.
- **Dynamic Web Interface:** A lightweight, non-blocking UI for fluid conversations.
- **Stateless Design:** Scalable backend architecture.
- **Error Resilience:** Global exception handling for consistent up-time.

## 🚀 Quick Start

### 1. Installation
```bash
npm install
```

### 2. Run the Engine
```bash
node server.js
```

### 3. Access the Terminal
Open your browser and navigate to `http://localhost:3000`.
